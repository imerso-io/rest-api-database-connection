# Passo a Passo de como criar o banco de dados utilizando DBeaver

### 1. Após entrar no DBeaver, vá em "Banco de dados" e clique em "Nova conexão"
![step1](https://github.com/inobrasil/restAPI-database-connection/assets/125743142/3a873b6f-2387-4437-b0cc-59b37697fe16)

### 2. Escolha o banco de dados "PostgreSQL" (Com a foto do elefante) e clique em "avançar"
![step2](https://github.com/inobrasil/restAPI-database-connection/assets/125743142/1419c1e7-188a-4c91-b52c-360f91becc5c)

### 3. Configure seu banco de dados
####    1 - Atualize a porta de "5432" para "5433"
####    2 - Atualize o nome do banco de dados de "postgres" para "users"
####    3 - Atualize a autenticação mantendo o nome de usuário, mas colocando a senha como "restAPI-ino"
####    4 - Clique em "concluir"
![step3](https://github.com/inobrasil/restAPI-database-connection/assets/125743142/577078a6-8ac4-4f43-845b-60f293dfaa07)

# Step by Step on how to create the database using DBeaver

### 1. After entering DBeaver, go to "Database" and click "New connection"
![step1](https://github.com/inobrasil/restAPI-database-connection/assets/125743142/3a873b6f-2387-4437-b0cc-59b37697fe16)

### 2. Choose the database "PostgreSQL" (With the picture of the elephant) and click "next"
![step2](https://github.com/inobrasil/restAPI-database-connection/assets/125743142/1419c1e7-188a-4c91-b52c-360f91becc5c)

### 3. Configure your database
####    1 - Update port from "5432" to "5433"
####    2 - Update the database name from "postgres" to "users"
####    3 - Update the authentication by keeping the username but putting the password as "restAPI-ino"
####    4 - Click "finish"
![step3](https://github.com/inobrasil/restAPI-database-connection/assets/125743142/577078a6-8ac4-4f43-845b-60f293dfaa07)
